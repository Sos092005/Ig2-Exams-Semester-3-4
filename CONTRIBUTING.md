# Contributeur externe
*Destiné aux étudiants souhaitant partager leurs épreuves d'examens*

1. Vous ouvrez simplement une [Issue](https://github.com/Sos092005/Ig2-Exams-Semester-3-4/issues) (un ticket) sur le dépôt
2. Vous y joignez votre scan pdf ou image
3. Et un collaborateur interne se chargera de l'intégrer correctement dans les archives.
